### 2. 显示键对应的符号

可以显示出符号和键盘的对应关系

|符号|键盘按键|
|:--|:--|
|⇥|tab |
|⇪ |caps lock |
|⌫| delete |
|⇧ | shift| 
|⌃  | control | 
|⌥ | option |
|⌘ | command |

图片如下：
![](/assets/image/blog/mac-keyboard/1.png)

### 3. 显示fn按下的符号

可以看出组合键的对应关系 `fn + `

|符号|键盘组合按键|说明|
|:--|:--|:--|
|⌦| `fn + delete`|向前删除一个内容|
|↑| `fn + ⬆︎`|page up|
|↓| `fn + ⬇︎`|page down|
|⤒| `fn + ⬅︎`|home|
|⤓| `fn + ➡︎`|end|
